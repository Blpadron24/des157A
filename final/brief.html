Game-On Redux

Project Summary

For this project I want to continue to work on my Game On Project. I want to add some features that make it feel more complete and give the user different variations of the game.

Project Goals

My goals for this project are to add a randomizer which chooses an image when the page is reloaded to change the plushie the players are playing for. That way the players are encouraged to play again for a different prize.

Design Strategies

The game is going to look the same, I may have an overlay open with the game rules so the players read them before moving on to play. I want to have the image displayed randomly when the page is loaded.

Experience Goals

I want to make sure the players of my game know how to interact with it easily. I want to give them enough feedback that makes them feel like they are interacting with the elements available. I also want them to want to play the game again which I think I can achieve by adding multiple prize options.
